# Extension Icons

To generate the required icon sizes from passman.png:

1. Use an online converter or imagemagick:

   ```bash
   convert passman.png -resize 16x16 icon-16.png
   convert passman.png -resize 48x48 icon-48.png
   convert passman.png -resize 128x128 icon-128.png
   ```

2. Or use an online tool like https://cloudconvert.com/svg-to-png

Required sizes:

- icon-16.png (16x16)
- icon-48.png (48x48)
- icon-128.png (128x128)
